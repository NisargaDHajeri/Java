# Mood-Based Personalized Music Player

A Java web application developed using JSP, Servlets, JDBC, and MySQL that allows users to upload, organize, search, and play songs based on mood categories like Happy, Chill, Sad, Party, and Romantic.

## Features
- Mood-based filtering
- Playlist management
- Favorites system
- Song upload
- Search functionality
- Music playback

## Technologies Used
- JSP
- Servlets
- MySQL
- JDBC
- Tailwind CSS
- Apache Tomcat
